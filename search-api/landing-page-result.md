# Busca por Landing Page

Esse tipo de busca é muito utilizado para a criação de Landing Pages customizadas. Para realizar esse tipo de busca, basta chamar o `SearchAPI` com o seguinte input

## Input

```json
{
  "token": "{{TOKEN_LOJA}}",
  "sort": "price.desc",
  "searchPath": "{{URL_PATH}}"
}
```
