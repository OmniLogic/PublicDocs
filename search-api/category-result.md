# Busca por Categoria

Esse tipo de busca é muito utilizado em páginas de departamentos ou categorias. Para realizar esse tipo de busca, basta chamar o `SearchAPI` com o seguinte input

## Input

```json
{
  "token": "{{TOKEN_LOJA}}",
  "sort": "price.desc",
  "categories": ["{{CATEGORIA_LOJA}}"]
}
```
